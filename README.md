E-commerce Data Analytics Project (SQL + Python)

This project presents an end-to-end data analytics workflow on a Brazilian e-commerce dataset containing over 100,000 orders. It demonstrates the integration of SQL and Python to perform data extraction, transformation, analysis, and visualization.

The dataset was imported into a MySQL database using Python (pandas and mysql-connector), enabling efficient querying and manipulation. SQL was used to perform exploratory and advanced analysis, including joins, subqueries, Common Table Expressions (CTEs), and window functions such as DENSE_RANK and LAG.

Python was utilized for data processing, correlation analysis (NumPy), and visualization (Matplotlib) to uncover meaningful business insights. Key analyses include revenue contribution by product category, seller performance ranking, customer purchasing behavior, moving averages, cumulative sales, and year-over-year growth.

This project highlights the practical application of SQL and Python in solving real-world business problems and showcases strong data analytics and storytelling skills.
